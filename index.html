ACADEMIC PROJECTS

Here are some of the projects I have completed as part of my academic coursework.

Job Recommender System

The project uses React , NodeJs , , Python , Machine Learning , and Natural Language Processing (NLP) to provide personalized job recommendations based on users' skills, experience, and career goals. The system analyzes large amounts of data, including resumes, job descriptions, and candidate profiles, to make accurate predictions about the best job matches for each user.

Spam Messages Detector

This application provides a simple and effective way to identify spam messages, using Machine Learning and Naive Bayes to classify messages based on their content. It is built using Python , and the Flask web framework, and leverages the power of machine learning to accurately identify spam messages.

Law Office Management System

Manages clients, cases, and billing records. The frontend is built with React , providing a user-friendly interface for managing these tasks. The backend is built with NodeJs , providing a server-side API for managing the data in the database. The database is MongoDB, which stores all the client information.
